NextJS in 7 minutes YouTube

NextJS is a React framework that handles:
- Routing
- Server Rendering
- API routes
- Server rendering
- SEO

Just start building after
```
npx create-next-app
```

`next.config.ts`: ranging from environment variables, to build instructions

Routing is done thru file system
Two types: App, Pages

**App Router**
    folder = page
    `about/` = site.com/about
**Pages Router**
    has a folder called `pages/`
    directly files inside, the name of the file will become the route 
    `about.tsx` = site.com/about

use app router. pages router is considered legacy.
