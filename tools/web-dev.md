**Vite**
HMR, builds code,  acts as server.

**npm**
Package manager for JS. Install and manage dependencies and JS libraries.

**HMR**
Hot module replacement. Reloads automatically for every file changes, no need for manual reloading. Like vscode live server extension.

**React**
Library for JS. Allow writing HTML in JS.

**Next.js**
Framework on top react, adds features like routing.

**Routing**
Decides what to show based on url.
Client side routing: JS swaps components, browser doesn't reload page.
Server side routing: Servers generates and return page based on url.
