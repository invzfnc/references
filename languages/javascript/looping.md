`while`
```js
while (condition) {
    ...
}
```

`do...while`
```js
do {
    ...
} while (condition);
```

`for`
```js
for (init; condition; step) {
    ...
}
```

`for...of`
```js
// foreach / for...in equivalent
// loops through each element in collection
for (const element of arr) {
	...
}
```

`break`: Breaks the loop
`continue`: Skips this iteration and continues to the next
