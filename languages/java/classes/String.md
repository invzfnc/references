```java
String s = "Hello, world!";

s.length()           // 13
s.charAt(5)          // ','
s.concat(" Again!")  // "Hello, world! Again!" (returns a new string)
s.toUpperCase()      // "HELLO, WORLD!"
s.toLowerCase()      // "hello, world!"
"  hi  ".trim()      // "hi" (trims whitespace characters)
```
